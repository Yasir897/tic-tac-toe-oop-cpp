# ⭕ Tic-Tac-Toe (C++ OOP)

A classic **Tic-Tac-Toe** game built in **C++** using Object-Oriented Programming — my **Object-Oriented Programming (OOP) project**.

## 🧩 Features
- Two-player Tic-Tac-Toe on a 3×3 board
- **OOP design** — separate classes for the game, board display, editing and play logic (with `friend` classes)
- Input validation and win/draw detection
- Clean console interface

## 🛠️ Tech
- **Language:** C++
- **Concepts:** Classes & objects, inheritance, friend classes, 2D arrays, encapsulation

## ▶️ Build & Run
```bash
g++ tic_tac_toe.cpp -o tictactoe
./tictactoe
```
